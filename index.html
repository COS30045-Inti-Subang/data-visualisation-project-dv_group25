Temporal Analysis

Seasonal Crash Patterns

This multi-series line chart reveals the seasonal patterns of road crashes in South Australia from 2020 to 2024. Each line represents a different crash type (Rear End, Hit Fixed Object, Right Angle), showing the average number of casualties per month. The visualization answers the question: "Is there a seasonal pattern to accidents?"

Safety Equipment Analysis

Seatbelt Impact on Injury Severity

This grouped bar chart examines the relationship between seatbelt usage and injury severity in South Australia road crashes from 2020 to 2024. The visualization compares injury outcomes (Fatal, Admitted to Hospital, Treated at Hospital, By Private) across different seatbelt usage categories, answering: "Does wearing a seatbelt reduce injury severity?"

Data-Driven Findings

Key Safety Insights

Our analysis of over 63,000 crash records and 23,000 casualty records from South Australia (2020-2024) reveals critical patterns in road safety. Seasonal trends show distinct crash patterns varying by type, while seatbelt usage demonstrates a measurable impact on injury severity. These insights guide targeted interventions, enforcement strategies, and public awareness campaigns to reduce road trauma across the state.

Data Methodology

Our analysis uses rigorous data processing workflows to ensure accurate insights

Data Processing Pipeline

1. Data Collection

Raw crash and casualty data sourced from South Australia Government Open Data Portal. Contains 63,241 crash records and 23,894 casualty records spanning 2020-2024.

2. Data Cleaning (KNIME)

KNIME workflows filter null values, standardize categorical variables, and remove incomplete records. This reduced crash data to 15,471 high-quality records.

3. Analysis & Visualization

D3.js processes cleaned data for aggregation and temporal analysis. Observable Plot renders interactive visualizations with real-time filtering capabilities.

4. Validation

Cross-referenced with official SA road safety reports. Statistical measures verified for consistency and reliability.
