Data Methodology

Our analysis uses rigorous data processing workflows to ensure accurate insights

Data Processing Pipeline

1. Data Collection

Raw crash and casualty data sourced from South Australia Government Open Data Portal. Contains 63,241 crash records and 23,894 casualty records spanning 2020-2024.

2. Data Cleaning (KNIME)

KNIME workflows filter null values, standardize categorical variables, and remove incomplete records. This reduced crash data to 15,471 high-quality records.

3. Analysis & Visualization

D3.js processes cleaned data for aggregation and temporal analysis. Observable Plot renders interactive visualizations with real-time filtering capabilities.

4. Validation

Cross-referenced with official SA road safety reports. Statistical measures verified for consistency and reliability.
